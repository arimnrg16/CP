Deskripsi

Nivri dan Ramma merupakan sepasang sahabat karib. Mereka juga senang berkompetisi. Alhasil, hampir setiap hari mereka bertanding, dari adu panco, bermain gundu, sampai adu ganteng (jangan tanya bagaimana caranya!). Suatu hari, mereka bertanding mengenai kecantikan string. Ramma memiliki string S, sedangkan Nivri memiliki string T. Keduanya memiliki panjang yang sama, dan hanya terdiri dari huruf kecil alfabet. Pertandingan ini berlangsung sangat alot, karena keduanya sama-sama merasa bahwa string miliknya lah yang paling cantik.

Siang hari itu, mereka sedang belajar tentang longest common subsequence (LCS). Subsequence dari suatu string merupakan hasil penghapusan 0, sebagian, atau semua karakter dari string tersebut, tanpa mengubah urutan. Contohnya, "rot" merupakan subsequence dari "ramot", namun "ar" bukan. LCS dari dua buah string merupakan subsequence terpanjang yang merupakan subsequence dari kedua string tersebut. Untuk mempermudah, didefinisikan LCS(x,y) sebagai panjang dari longest common subsequence string x dan y.

Pertandingan Nivri dan Ramma ini membuat sensei (guru) mereka kesal. Agar mereka dapat akur kembali, sensei lalu memberikan PR berikut:

Buatlah sebuah string R. String tersebut harus memiliki panjang yang sama dengan S dan T, dan hanya terdiri dari huruf kecil alfabet. Selain itu, LCS(R,S) + LCS(R,T) harus maksimal.

Nivri dan Ramma pun lupa dengan pertandingan mereka karena PR tersebut terlalu sulit! Sekarang, mereka justru membuat PR tersebut sebagai pertandingan mereka, dengan pemenangnya adalah yang menemukan jawabannya terlebih dahulu. Karena tidak ingin kalah dari Nivri, Ramma lalu meminta bantuanmu. Bantulah Ramma!

Format Masukan

Baris pertama berisi sebuah string S, string milik Ramma.

Baris kedua berisi sebuah string T, string milik Nivri.

Format Keluaran

Satu baris berisi sebuah string R, jawaban untuk PR dari sensei. Jika ada beberapa R yang memenuhi, keluarkan yang mana saja.

Contoh Masukan

ddadbebe
ecbdccae

Contoh Keluaran

eddccabe

Penjelasan

  • LCS("eddccabe","ddadbebe") = 5, contoh subsequence-nya adalah "ddabe".
  • LCS("eddccabe","ecbdccae") = 6, contoh subsequence-nya adalah "edccae".

Sehingga, totalnya adalah 11. Selain "eddccabe", masih banyak string R yang menyebabkan LCS(R,S) + LCS(R,T) = 11, seperti "dbdcadbe" dan "dddbebce". Tidak ada R', sehingga LCS(R',S) + LCS(R',T) > 11.

Batasan

  • 1 ≤ |S| = |T| ≤ 300
  • Karakter dari S dan T hanya terdiri dari huruf kecil alfabet.
