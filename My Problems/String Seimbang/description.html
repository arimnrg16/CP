Deskripsi

Pak Chanek senang membaca soal-soal Competitive Programming. Suatu hari, ia membaca topik tentang tanda kurung. Pada topik tersebut, diperkenalkan istilah seimbang. Penasaran, Pak Chanek membacanya lebih lanjut. Ternyata, sebuah string yang hanya terdiri dari karakter ( dan ) dikatakan seimbang, jika memenuhi salah satu dari 3 bentuk berikut:

  1. String tersebut berupa ().
  2. String tersebut berupa (A), dengan A suatu string seimbang.
  3. String tersebut berupa AB, dengan A dan B suatu string seimbang.

Untuk menguji kemampuannya, Pak Chanek lalu membuat suatu string S, yang hanya terdiri dari karakter ( dan ), serta memiliki panjang N. Lalu, ia menghitung banyaknya substring-nya yang seimbang. Namun, karena ukurannya yang cukup besar, akhirnya Pak Chanek kewalahan! Bantulah Pak Chanek menyelesaikan persoalan ini.

Suatu string U dikatakan substring dari V apabila terdapat (i,j), 1 ≤ i ≤ j ≤ N, sehingga U = V[i..j]. Dua buah substring, masing-masing pada rentang [a,b] dan [c,d], dikatakan berbeda apabila a ≠ b atau c ≠ d.

Format Masukan

Baris pertama berisi sebuah bilangan bulat N, panjang dari string milik Pak Chanek.

Baris kedua berisi sebuah string S, string milik Pak Chanek.

Format Keluaran

Keluarkan satu baris berisi sebuah bilangan bulat, banyaknya substring dari S yang merupakan string seimbang.

Contoh Masukan 1

6
(()())

Contoh Keluaran 1

4

Contoh Masukan 2

5
())()

Contoh Keluaran 2

2

Penjelasan

Pada contoh masukan 1, substring yang merupakan substring seimbang adalah (), (), ()(), dan (()()). Perhatikan bahwa () dihitung dua kali karena muncul pada dua tempat berbeda.

Subsoal

Pada setiap subsoal berlaku:

  • 1 ≤ N ≤ 1.000.000
  • S hanya terdiri dari karakter ( dan )

Subsoal 1 (16 poin)

  • 1 ≤ N ≤ 200

Subsoal 2 (23 poin)

  • 1 ≤ N ≤ 2.000

Subsoal 3 (11 poin)

  • Terdapat i, dengan 1 ≤ i ≤ N, sehingga Sa = Sa-1 untuk 1 < a ≤ i dan Sb = Sb+1 untuk i < b < N

Subsoal 4 (11 poin)

  • S_i ≠ Si-1 untuk 1 < i ≤ N

Subsoal 5 (39 poin)

  • Tidak ada batasan khusus
