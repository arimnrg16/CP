Jawaban yang betul adalah:

SEMUA COWOK PASTI KELEPEK-KELEPEK SAMA CEWEK YANG
C O D I N G

Mungkin beberapa dari Anda sudah tahu teka-teki nyeleneh Cak Lontong seperti teka-teki tersebut. Karena sekarang sudah memasuki bulan puasa, maka Cak Lontong ingin memberikan teka-teki sedemikian rupa sehingga tidak ada peserta yang menjadi emosi (karena nyeleneh tersebut).

Cak Lontong mengetahui bahwa jawaban dari sebuah teka-tekinya adalah string S. Anda akan diberikan sebuah string S' yang karakter-karakternya terdiri dari huruf kecil, namun bisa jadi beberapa karakter tersebut bukanlah huruf kecil melainkan berupa garis bawah (_). Setiap garis bawah masing-masing mewakili sebuah huruf kecil. Anda diminta untuk mencari tahu string jawaban S dari string S' tersebut. Cak Lontong juga memberikan sebuah bilangan bulat K. Cak Lontong kemudian memberikan petunjuk berikut:

Apabila Anda mengurutkan seluruh kemungkinan string yang mungkin dibentuk dari string S' secara leksikografis, maka string jawaban S berada di urutan ke-K.

Dapatkah Anda menemukan string jawaban S dari petunjuk yang diberikan Cak Lontong?

Format Masukan

Baris pertama berisi sebuah string S'.

Baris kedua berisi sebuah bilangan bulat K.

Format Keluaran

Keluarkan sebuah baris berisi string S dari petunjuk yang diberikan Cak Lontong.

Contoh Masukan 1

caklon_ong
10

Contoh Keluaran 1

caklonjong

Contoh Masukan 2

c_____
6456145

Contoh Keluaran 2

coding

Penjelasan

Pada contoh 1, terdapat 26 kemungkinan jawaban yang apabila diurutkan secara leksikografis adalah:

  • "caklonaong"
  • "caklonbong"
  • "cakloncong"
  • ...
  • "caklonyong"
  • "caklonzong"

String S adalah string pada urutan ke-10 yakni "caklonjong".

Batasan

  • 1 ≤ |S'| ≤ 100
  • 1 ≤ K ≤ 265
  • Banyaknya karakter garis bawah pada S' antara 0 hingga 5 inklusif
  • Karakter-karakter pada S' hanya terdiri dari huruf kecil atau garis bawah
  • Nilai K dijamin tidak lebih dari banyaknya kemungkinan jawaban (dengan kata lain, selalu ada jawaban)
